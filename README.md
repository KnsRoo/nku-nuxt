# NKU

## Build Setup

```bash
# install dependencies
$ npm install

# serve with hot reload at localhost:3000
$ npm run dev

# build for production and launch server
$ npm run build
$ npm run start

# generate static project
$ npm run generate
```
```javascript
# to enable ssr, in nuxt.config.js:
ssr: true
# target directory:
generate: {
  dir: "public" 
}
```
